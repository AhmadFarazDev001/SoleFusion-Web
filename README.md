## 🚀 Overview

**SoleFusion** is an interactive, multipage e-commerce and brand landing experience built from the ground up for ultimate performance and stunning visual fidelity. Engineered by Ahmad Faraz with the help of AI (aka **Vibe Coding**), the application focuses heavily on high-end animation, smooth navigation, and a dynamic 3D-feeling UI.

## ✨ Core Features

- **Fluid Scroll Architecture**: Powered by GSAP and Framer Motion, providing seamless, physics-based scroll triggers and layout morphs.
- **Dynamic Contextual Coloring**: The global aesthetic, background gradients, and ambient lighting of the entire application uniquely adapt to whatever model of shoe is currently active in the user's viewport.
- **Glassmorphism & 3D Elements**: Uses custom backdrop filters, drop-shadows, and `transformPerspective` manipulation to make products leap off the screen.
- **Custom Audio Engine**: Uses a bespoke HTML5/Web Audio API synthesizer to execute precisely timed impact sounds perfectly aligned with user interactions.
- **Seamless Page Routing**: Zero page-load flashes. Clicking any navigation element unmounts and remounts views (Home, Archive, Tech, About) with buttery-smooth `<AnimatePresence>` transitions.
- **Responsive Mastery**: Carefully tuned breakpoints ranging from strict `<768px` mobile clamps to expansive `1440px+` ultra-wide container sizing to ensure zero clipping and massive product impact.
- **Hardened Application Flow**: Outfitted with anti-inspect scripts intercepting standard browser developer tools and right-click menus directly in the DOM.

## 🛠 Tech Stack

- **Framework**: React 19 + TypeScript
- **Bundler / Dev Environment**: Vite
- **Styling**: Tailwind CSS v4 + Native CSS variables
- **Animation Integration**: GSAP (GreenSock) & Framer Motion
- **Icons**: Lucide React

## 📦 Getting Started

**Prerequisites:** You must have Node.js installed on your machine.

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the local development server:**
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the localhost port provided in your terminal (usually `http://localhost:3000` or `http://localhost:5173`).
